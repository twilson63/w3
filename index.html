w3

A static web server that is easy to install and easy to run.

It also comes with pushState support so any non asset request will return index.html.

Install

npm install w3 -g

Usage

w3

or

w3 [port]

options

Specify target root directory

--directory [dir]

w3 --directory www

Enable pushstate

--pushstate

w3 --pushstate
